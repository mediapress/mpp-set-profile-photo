== Description ==

An add-on for MediaPress. Allow users to set MediaPress Photo as BuddyPress profile picture.

== Installation ==

This section describes how to install the plugin and get it working.

You may need to follow the following steps.

1. Download `mpp-set-profile-photo.xyz.zip` (where xyz is version number )and extract
2. Upload `mpp-set-profile-photo` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. That's all.
